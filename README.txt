== Description ==

The **Custom Login Template** allows WordPress site administrators to select from a variety of pre-built login page templates, providing an easy way to customize the login experience without coding. 

With this plugin, you can:
- Choose from a variety of customizable login page templates.
- Change the look and feel of your login screen with a few clicks.
- Manage templates directly from the WordPress dashboard.
- Easily switch between templates without affecting your site's functionality.

Whether you want a simple login screen or a more complex, styled design, this plugin makes it easy to tailor the user experience to match your brand.

== Features ==

- Select and apply pre-designed login page templates.
- No coding required to customize the look of your login screen.
- Choose templates from the plugin settings page in the WordPress dashboard.
- Easily switch between templates.
- Lightweight and easy to use.

== Installation ==

1. Upload the `Custom Login Template` folder to your WordPress `wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the plugin settings under **Settings > Login Template** to choose your preferred login template.
4. Save your settings to apply the new template.

== Frequently Asked Questions ==

= How do I change the login page template? =

Once you activate the plugin, go to **Settings > Login Template** and choose your preferred template. After saving your settings, the new login page template will be applied.

= Can I add custom templates? =

Currently, the plugin supports a selection of pre-built templates, but you can extend it by adding custom templates to the plugin directory.

= Is this plugin compatible with other login customization plugins? =

It is compatible with most login customization plugins. However, certain features might conflict with other login page customizations, so we recommend testing it in a staging environment before using it on your live site.
